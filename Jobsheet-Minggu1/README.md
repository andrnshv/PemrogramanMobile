Setup Awal Android Studio dan Flutter
1. Download dan instalasi Android Studio
2. Download Flutter dan extract folder
3. Setting PATH Flutter ke dalam environment variable agar terbaca di CMD
4. Memastikan PATH untuk Flutter terbaca di CMD dengan menjalankan flutter --version
5. Selanjutnya memverifikasi Flutter dengan menjalankan flutter doctor
6. Verifikasi Lisensi flutter dengan menjalankan flutter --android-licenses
7. Install Flutter pada ekstensi VSCode
8. Jalankan Dart application pada VSCode
9. Jalankan Dart Hello World menggunakan command Dart run
10. Modifikasi Aplikasi Dart dengan mengubah menjadi rumus segitiga
