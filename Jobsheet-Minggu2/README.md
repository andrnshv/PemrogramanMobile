Laporan Tugas Pemrograman Dart

Dokumentasi ini berisi penjelasan mengenai program Dart yang telah dibuat untuk memenuhi kriteria Tugas Individual dan Tantangan Tambahan.

Rincian Program:

Penghitung BMI (Body Mass Index)
Program untuk menghitung indeks massa tubuh dengan input berat badan (kg) dan tinggi badan (cm). Program menentukan apakah pengguna termasuk kategori kurus, ideal, atau obesitas.

Kalkulator Mata Uang
Program konversi nilai mata uang dari Rupiah (IDR) ke tiga mata uang asing: USD (Dollar Amerika), SGD (Dollar Singapura), dan JPY (Yen Jepang).

Implementasi Operator
Aplikasi konsol yang mendemonstrasikan seluruh jenis operator dalam Dart, mulai dari operator aritmatika, perbandingan, logika, hingga assignment dengan input dinamis.

Tantangan Tambahan: Konversi Unit
Aplikasi konversi satuan yang mencakup kategori Panjang, Massa, Volume, dan Suhu. Program ini menggunakan Map untuk menyimpan faktor konversi dan memiliki validasi agar nilai input tidak negatif.
